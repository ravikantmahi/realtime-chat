# Real-Time Chat App (Frontend)

This is the React frontend for a real-time chat application. It connects to a Node.js/Socket.io backend to enable instant, bidirectional messaging.

## Features
* **Real-time Messaging:** Instant message delivery using WebSockets.
* **Typing Indicators:** See when the other user is typing with animated UI.
* **Smooth UI/UX:** Features CSS entrance animations and smooth auto-scrolling.
* **Responsive Design:** Clean, mobile-friendly chat interface.

## Tech Stack
* React.js
* Socket.io-client
* CSS3 (Flexbox & Keyframe Animations)

## Running Locally

1. Clone the repository.
2. Install dependencies:
   \`\`\`bash
   npm install
   \`\`\`
3. Start the development server:
   \`\`\`bash
   npm start
   \`\`\`
*Note: This frontend requires the backend server to be running concurrently to function properly.*
